HCI Lab - Comparison of Keyboard Layouts

Comparison between QWERTY and DVORAK Keyboard Layouts

QWERTY Text Area

<!-- HTML -->
			<textarea class="qwerty"></textarea>
		// Script
		$('.qwerty:eq(2)')
		 .keyboard({
		  layout   : 'qwerty',
		 })
		 .addTyping();

Dvorak

<!-- HTML -->
		<textarea id="dvorak"></textarea>
		// Script
		$('#dvorak')
		 .keyboard({ layout: 'dvorak' })
		 .addTyping();
